The constant folder stores all of the constants for the program. They should never be edited
Most constants are set in the Kingdom Of Avarya Rules(ex: Bard gets 25% increase taxable income on entertainment)

The bonus file stores all country bonuses player's get from their class. 
The file is organized by dictionaries of dictionaries. 
Each class has a BONUSES entry, which stores what type of bonuses each class gets.
The INATE entry stores what bonuses each province recieve automaticly.
The INDEPTH entry stores what bonuses a province gets in the player works in depth.

A list of all the different types of bonuses for classes
  INATE bonuses
    SATISFACTION_LOWER_CLASS - How much bonus satisfaction the lower class gets (int)
    SATISFACTION_MIDDLE_CLASS - How much bonus satisfaction the middle class gets (int)
    SATISFACTION_UPPER_CLASS - How much bonus satisfaction the upper class gets (int)
    ENTERTAINMENT_PRODCUTION - A percent increase to the taxable income generated by entertainment districts (decimal)
    LUMBER_PRODCUTION - A percent increase to the taxable income and resources generated by lumber districts (decimal)
    LIVESTOCK_PRODCUTION - A percent increase to the taxable income and resources generated by livestock districts (decimal)
    CROP_PRODCUTION - A percent increase to the taxable income and resources generated by crop districts (decimal)
    WATER_PRODCUTION - A percent increase to the taxable income and resources generated by water districts (decimal)
    PIETY_SPECIFIC - How much bonus piety is a certain god recieves (int)
    PIETY_GENERAL - How much bonus piety gods of a certain allignment (int)
    DEFENSE - How much bonus defense each province gets (int)
    MILITARY_TRAINING - The amount of seasons training units is reduced by (int)
    TRADE_SPEED - A percent reduction to how long it takes companies to trade (decimal)
    TRADE_PROFIT - A percent increase to how much profit is gained from companies (decimal)
  
  INDEPTH bonuses
    DEFENSE - How much bonus defense the province gets (int)
    SATISFACTION - How much bonus satisfaction each class gets in the province (int)
    LUMBER_ADV - Province gets advantage on lumber production check (bool)
    CROP_ADV - Province gets advantage on crop production check (bool)
    LIVESTOCK_ADV - Province gets advantage on livestock production check (bool)
    MINERAL_ADV - Province gets advantage on mineral production check (bool)
    WATER_ADV - Province gets advantage on water production check (bool)
    PRODUCTION_ADV - How many bonus production checks the province gets advantage on (int)

The bonus file also stores all bonuses a species provides to a province, as well as a species population rate.
These are also organized as dictionaries of dictionaries.
Each of the species has a BONUSES entry, which stores what type of bonuses each specie gets.
The PRIMARY_BONUS entry stores what the specie provides when it is the most populous specie in a province
The SECONDARY_BONUS entry stores what the specie provides when it is the second most populous specie in a province
The COMBAT_BONUS entry stores what the specie provides when it is the most populous specie in a army unit
